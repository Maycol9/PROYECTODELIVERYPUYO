SOBRE EL PROYECTO

¿Qué es DeliverPuyo?

DeliverPuyo es una aplicación web académica desarrollada para la gestión de usuarios, productos e inventario, con una interfaz moderna y una estructura organizada.

Propósito del sistema

El proyecto nace con la finalidad de construir una solución web funcional capaz de administrar el catálogo de productos, el acceso de usuarios y el control de información de manera clara, segura y profesional.

  • Gestión centralizada de productos
  • Autenticación y control de acceso
  • Visualización clara de inventario
  • Interfaz adaptada a una experiencia moderna

Tecnologías utilizadas

Python Flask Jinja2 Flask-Login SQLite MySQL HTML CSS

La arquitectura del sistema está pensada para combinar backend, persistencia de datos y diseño frontend dentro de un mismo proyecto.

Alcance funcional

DeliverPuyo integra componentes clave para el desarrollo de una aplicación web: autenticación, administración de productos, control de inventario, panel principal y organización de la información del sistema.

Valor del proyecto

Más que una página visualmente atractiva, DeliverPuyo representa una implementación práctica de conceptos de desarrollo web, estructura de datos, persistencia y experiencia de usuario.
